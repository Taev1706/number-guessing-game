# Number Guessing Game — Угадай число

**Версия:** 1.0.2  
**Язык:** Python 3.10+  
**Хранение данных:** CSV (встроенный модуль `csv`)  
**Тестирование:** pytest + pytest-cov

---

## Описание

Консольная игра: компьютер загадывает случайное число от 1 до 100, игрок угадывает, получая подсказки «больше» или «меньше». Ведётся таблица лучших результатов (имя игрока + количество попыток).
Реализует полный цикл разработки: от ТЗ и TDD до приёмочного тестирования и поддержки.

## Инструкция по запуску

```bash
# 1. Клонировать репозиторий
git clone https://github.com/your-username/number-guessing-game.git
cd number-guessing-game

# 2. Создать виртуальное окружение (рекомендуется)
python -m venv venv
venv\Scripts\activate      # Windows
source venv/bin/activate   # Linux/Mac

# 3. Установить зависимости для тестов
pip install pytest pytest-cov

# 4. Запустить игру
python menu.py

# 5. Запустить тесты
pytest tests/ -v

# 6. Проверить покрытие кода
pytest tests/ --cov=data_service --cov=logic --cov-report=term-missing
```

## Пользовательские истории (User Stories)

| # | История |
|---|---------|
| US-01 | Как игрок, я хочу запустить игру и ввести своё имя, чтобы мой результат сохранился |
| US-02 | Как игрок, я хочу получать подсказки «больше» / «меньше», чтобы быстрее угадать число |
| US-03 | Как игрок, я хочу видеть, сколько попыток мне понадобилось, чтобы знать свой результат |
| US-04 | Как игрок, я хочу посмотреть таблицу рекордов, чтобы сравнить себя с другими |
| US-05 | Как игрок, я хочу получать понятные сообщения об ошибке при некорректном вводе |

## MVP (Минимально работающий продукт)

- Компьютер загадывает случайное число 1–100 (US-01)
- Игрок вводит имя (US-01)
- Подсказки «больше» / «меньше» (US-02)
- Подсчёт и вывод количества попыток (US-03)
- Сохранение результата в CSV (US-03)
- Просмотр таблицы рекордов топ-10 (US-04)
- Валидация ввода: нечисловое, вне диапазона, пустое (US-05)

## FAQ

**Q: Как сбросить все рекорды?**  
Удалите файл `data/records.csv` или выполните:
```python
python -c "import data_service; data_service.clear_all()"
```

**Q: Как перенести рекорды на другой компьютер?**  
Скопируйте файл `data/records.csv` в папку с игрой на другом компьютере.
